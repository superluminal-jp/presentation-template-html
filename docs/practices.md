# 採用ベストプラクティスと出典(practices.md)

本テンプレートが埋め込むプラクティスの**正本**。各スライドの注釈(`data-practice`)と巻末リファレンス(`slides/08-reference.html`)は、ここに定義した **ID** を参照する(重複を避けるため書誌の直書きはしない)。5 群(A〜E)で整理し、粒度は「プレゼン全体(A)」「1ページ内(B)」と横断群(C/D/E)を網羅する。

> 誠実利用の原則: 説得系(群 E 一部)は事実に基づく強調・理解促進にのみ用い、誇張や誘導的操作には用いない。
