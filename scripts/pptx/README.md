# PowerPoint エクスポート(scripts/pptx)

ショーケースデック(`index.html`)を、**ネイティブに編集可能な `.pptx`** へ書き出す Node 実装。テキストボックス・図形・ネイティブ表のみで構成し、本文領域にラスタ画像を使わない。

## 使い方

```bash
npm run build:pptx      # -> dist/sample-deck.pptx
```

`dist/` は Git 管理外。生成物は PowerPoint / Keynote / Google スライドで開ける。

## 構成

| ファイル | 役割 |
|---|---|
| `tokens.mjs` | デザイントークンの解決値(色 / フォント / サイズ)。`styles/tokens.semantic.css` を `tokens/vendor/tokens.css` 経由で解決した hex を集約(手維持) |
| `build-pptx.mjs` | スライド生成器。`buildDeck()` が全スライドを組み立て、共通チャーム(機密区分 / 著作権 / ページ番号のプレースホルダ)を各スライドに付与 |

## 出力内容(20 スライド)

- **コア 16 レイアウト**: タイトル / 目次 / 章扉 / 本文 / 比較 / 図表 / まとめ / ビッグナンバー / ダッシュボード / タイムライン / マトリクス / プロセス / 引用 / 全面画像 / クロージング / 出典。
- **付録 4 枚**: 状態と通知(コールアウト / バッジ / タグ / ボタン / 区切り)/ 告知と手順(バナー / ステップナビ)/ 指標とまとめ(**ネイティブ表 `addTable`** / プログレス / 統計 / チェックリスト / kv)/ DADS 公式(カード / チェックボックス / ラジオ / パンくず)。

## 方針・整合

- キャンバス 1280×720 px = 13.333×7.5 in(16:9)。px→pt は ×0.75、px→in は ÷96。
- 配色はトークン解決値に一致(accent `#0017C1` ほか)。テーマフォントは Noto Sans JP。
- 図表スライドとダッシュボード / グラフ系の SVG は簡略化または省略(コンポーネントの体裁が伝わることを優先)。

## 制約(既知)

- **幾何は近似**(HTML とのピクセル一致は非対象)。CSS Grid 座標の自動解析はせず、位置は手指定。
- **フォント埋め込みは未対応**(pptxgenjs 非対応)。未導入端末では代替表示。
- **`.potx` テンプレート化は未対応**(空テンプレート化は将来課題)。

対応スペック: [specs/007-appendix-consolidation](../../specs/007-appendix-consolidation/)(付録の PPTX 反映)。
